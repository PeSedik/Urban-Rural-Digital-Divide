# Urban-Rural Digital Divide in Europe
This project analyzes the digital divide between urban and rural areas in Europe using Eurostat data from 2007 to 2019. The analysis focuses on the digital transformation over time and provides insights into the disparities in internet access.

## Files
- **DV.pbix**: Power BI file containing the visualizations and analysis.
- **isoc_ci_in_h_custom_19042023.xlsx**: Excel data file from Eurostat, containing internet access data by territorial category.

## How to Reproduce
1. Download the files.
2. Open the `.pbix` file in Power BI.
3. Follow the steps in the Power BI file to view the visualizations and insights.
